# SpeechParse

**SpeechParse** — это приложение для распознавания русской речи с микрофона или из аудиофайлов.  
Программа записывает звук, преобразует его в текст и отображает результат в графическом интерфейсе.

## Возможности

- Запись аудио через микрофон с возможностью паузы и возобновления.  
- Распознавание речи на русском языке в реальном времени.  
- Загрузка и распознавание аудиофайлов (WAV, MP3, OGG).  
- Копирование текста в буфер обмена и сохранение в `.txt` файл.  
- Очистка текстового поля.  
- Отображение состояния приложения: запись, пауза, обработка, ожидание.

## Структура проекта

- **audio_handler.py** — работа с микрофоном и аудиофайлами.  
- **recognizer_vosk.py** — распознавание речи через Vosk.  
- **controller.py** — управление потоками, состояниями и текстом.  
- **gui.py** — графический интерфейс на Tkinter.  
- **main.py** — точка входа в приложение.

## Установка и запуск

1. Клонируйте репозиторий:  
```
git clone https://github.com/adaravaks/SpeechParse.git
```

2. Установите зависимости:  
```
pip install -r requirements.txt
```
(включает `sounddevice`, `pydub`, `vosk`, `tkinter` и `pyperclip`)

3. Скачайте русскую модель Vosk и распакуйте её в папку проекта, например `vosk-model-small-ru-0.22`.

4. Запустите приложение:  
```
python main.py
```

## Использование

- **Record** — запуск записи аудио.  
- **Pause / Resume** — пауза и возобновление записи.  
- **Stop** — остановка записи и запуск распознавания.  
- **Open File** — выбор аудиофайла для распознавания.  
- **Copy Text** — копирование текста в буфер.  
- **Save Text** — сохранение текста в файл.  
- **Clear Text** — очистка текстового поля.
